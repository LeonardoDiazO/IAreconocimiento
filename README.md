# IA Vision Pro 🚀

IA Vision Pro es una aplicación web moderna de reconocimiento de objetos en tiempo real que utiliza Inteligencia Artificial (**TensorFlow.js**) para identificar elementos a través de la cámara web con una interfaz premium y minimalista.

![Dashboard Preview](file:///C:/Users/Maximun_neo/.gemini/antigravity/brain/55e3c126-6d92-40eb-a1f2-b0e4fa26de08/main_dashboard_ia_vision_pro_1772424915794.png)

## ✨ Características Principales

- **Detección en Tiempo Real**: Identificación instantánea de múltiples objetos usando el modelo **Coco-SSD**.
- **Interfaz Premium**: Diseño basado en *Glassmorphism* con tema oscuro y tipografía refinada (Inter).
- **Controles Avanzados**:
  - ⏹️ **Control de Cámara**: Inicia o detén el stream de video fácilmente.
  - 📸 **Captura Inteligente**: Descarga instantáneas de lo que ves con las etiquetas de IA integradas.
  - 🎚️ **Sensibilidad Ajustable**: Controla el umbral de confianza para filtrar detecciones.
  - 📜 **Historial de Detección**: Listado dinámico de los objetos identificados recientemente.
- **Arquitectura Moderna**: Construido sobre **Vite** para un rendimiento óptimo.

## 🛠️ Tecnologías

- [TensorFlow.js](https://www.tensorflow.org/js) - Motor de IA.
- [Vite](https://vitejs.dev/) - Entorno de desarrollo y empaquetado.
- [Inter Font](https://fonts.google.com/specimen/Inter) - Tipografía premium.
- Vanilla CSS & Javascript (ES6+).

## 🚀 Instalación y Uso

### Requisitos Previos

- [Node.js](https://nodejs.org/) (Versión 18 o superior recomendada).

### Pasos

1. **Clonar el repositorio**:
   ```bash
   git clone <url-del-repositorio>
   cd IAreconocimiento
   ```

2. **Instalar dependencias**:
   ```bash
   npm install
   ```

3. **Ejecutar en desarrollo**:
   ```bash
   npm run dev
   ```

4. **Acceder a la aplicación**:
   Abre tu navegador en `http://localhost:5173` (o el puerto indicado en la terminal).
